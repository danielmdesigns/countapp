# Count App

The perfect mini calculator for all your devices. Count saves you time and effort, offering an attractive and beautiful interface, along with providing easy to use solutions for quick and convenient work with numbers.

Add Count to your home screen and/or save to your browsers favorites bar to use it any time.

Count was built using HTML, SASS, Flexbox, JavaScript, jQuery, fastClick.js and has 10-key functionality

Based on the FreeCodeCamp challenge Build A JavaScript Calculator (https://www.freecodecamp.com/challenges/build-a-javascript-calculator)

Enjoy!
